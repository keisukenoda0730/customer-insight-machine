基本的な使い方
1. サイドバーを確認
APIキーと検索エンジンIDが自動入力されていればOKです。

2. キーワードを入力
「🎯 検索キーワード」欄に悩みに関するワードを入力します。


例: ダイエット / 婚活 / 副業 / 英語学習
3. 設定を確認
設定	推奨	説明
🔥 深掘り検索モード	ONのまま	「辛い」「失敗」等を自動掛け合わせ
📊 ベース取得件数	10件から試す	APIを10件/クエリ消費する
📄 本文の最大文字数	2000字のまま	NotebookLM用に適切なサイズ
⚠️ 深掘りモードON・10件設定の場合、6クエリ × 1ページ = 6回のAPI消費です

4. 「🚀 リサーチ開始」をクリック
処理の流れ：

Google検索（数秒）
各URLの本文取得（1件ごとに3秒待機するので、20件なら約1分かかります）
5. 結果を確認・ダウンロード
完了後に2つのタブが表示されます：

📋 一覧プレビュー → 取得内容をその場で確認
📥 エクスポート → ダウンロードボタンが2つ
📊 CSV → ExcelやGoogleスプレッドシートで開く用
🤖 Markdown → NotebookLMにそのままアップロードする用
はじめての方におすすめの手順
キーワード: ダイエット
深掘りモード: ON
件数: 10件（API消費を抑えて動作確認）
「リサーチ開始」→ 結果が出たらMarkdownをダウンロード
NotebookLM を開いてMarkdownをアップロード → 自動で分析開始
